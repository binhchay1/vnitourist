The only plugin that allows you to easily create a timed full-screen ad.

== Description ==

The "Full screen ad" plugin helps you increase the earnings and conversions of your WordPress website.

Easily create full screen ads to show to your website visitors.

You can set the frequency and the waiting time of the announcement when it is shown.

Several filters are available to customize the functions of the plugin, for example it is possible to modify the list of bots to be excluded from the display of the ad.

The [developer](https://www.andreadegiovine.it/?utm_source=wordpress_org&utm_medium=plugin_page_text&utm_campaign=full_screen_ad "developer") works to improve the performance of the plugin, if you like it leave a [5 star review](https://wordpress.org/support/plugin/full-screen-ad/reviews/?filter=5 "Review this plugin") or [send a donation](https://www.andreadegiovine.it/outlinks/1422/?utm_source=wordpress_org&utm_medium=plugin_page&utm_campaign=full_screen_ad "send donation"), if you want to report a bug visit the support section.


== Full screen ad development ==

Using the "Full screen ad" plugin you will have some filter that you can use:

* apply_filters('full_screen_ad_bot', 'bot|crawl|spider|mediapartners|slurp|patrol')

Do you need other functions? Ask the developer from the [support](https://wordpress.org/support/plugin/full-screen-ad/ "support") section.

== Installation ==

To **automatically install** the "Full screen ad" plugin you can search from the "Plugins > Add new" section of your WordPress dashboard and click on the "Install Now" button corresponding to this plugin.

To **manually install** the "Full screen ad" plugin you can download the latest version from the WordPress.org site, and unpack the zip file in the "wp-content/plugins" folder of your CMS.

Now on the "Plugins" section of your WordPress dashboard you can see the "Full screen ad" plugin, click on "Activate".
